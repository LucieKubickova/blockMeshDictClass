# blockMeshDictClass
Python class for creation of blockMeshDict for OpenFOAM utility blockMesh.

## Compatability
Prepared for python3 (https://www.python.org/downloads/release/python-31010/) and OpenFOAMv8 (https://openfoam.org/version/8/).
